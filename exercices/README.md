# Exercices

Ce dossier contient les fichiers de départ pour les exercices pratiques du Module 0.

## Convention

Chaque chapitre qui a un exercice pratique dispose d'un sous-dossier :

- `chapitre-02/` — Exercices de manipulation du terminal
- `chapitre-05/` — Exercices de Git
- `chapitre-06/` — Exercices avec Claude Code

## Comment les utiliser

Les énoncés des exercices sont **dans les chapitres eux-mêmes** (dans `../chapitres/`). Ce dossier contient seulement les fichiers nécessaires à certains exercices, et c'est aussi l'endroit où tu vas créer tes propres fichiers d'exercice au fur et à mesure.

## Tes propres fichiers

N'hésite pas à créer ici tes propres notes, tests, et brouillons. Le `.gitignore` du repo est configuré pour ne pas tracker les fichiers personnels nommés `notes-perso-*.md` si tu veux garder certaines choses en local sans les pousser.
