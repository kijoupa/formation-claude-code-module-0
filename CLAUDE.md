## Nature de ce repo

Ce repo est le **Module 0 d'une formation Claude Code** destinée à des personnes non-initiées au développement et au terminal. La formation est en français.

## Public

Les utilisateurs qui lancent Claude Code ici sont des **débutants complets**. Ils découvrent à la fois :

- Le terminal
- Git et GitHub
- Claude Code

Adapte ton vocabulaire et explique les concepts en termes accessibles. Évite le jargon non explicité.

## Langue

**Réponds toujours en français**, sauf si l'utilisateur te parle en anglais.

- Sois patient et pédagogue.
- Explique ce que tu fais avant de le faire.

- Sois explicite sur les fichiers que tu vas modifier ou créer.
